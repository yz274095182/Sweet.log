# 🎨 专属控糖手记 UI/UX 详细设计文档 (完整版)

## 一、 整体视觉与色彩基调 (Design Tokens)
抛弃医疗 App 常用的“医院蓝”和“警报红”，采用“低饱和度暖色系”，营造治温、充满安全感的氛围。

* **🟢 主色 (Primary)：** `brand-orange: #FF9F43` (用于主按钮、核心高亮)
* **⚪ 背景 (Background)：**
  * 页面底色：`bg-base: #F8F9FA` (米灰色，降低视觉疲劳)
  * 卡片底色：`bg-card: #FFFFFF` (纯白，配合微弱弥散阴影 `shadow-sm` 形成悬浮感)
* **🚦 状态 (Status)：**
  * 达标 (Success)：`status-green: #52C41A` (莫兰迪绿)
  * 偏高 (Warning)：`status-warning: #FA8C16` (柔和橙)
  * 偏低 (Danger)：`status-danger: #F5222D` (醒目红，低血糖极度危险)
* **✒️ 字体 (Typography)：**
  * 标题/数字：`text-main: #333333`
  * 正文/副标：`text-sub: #666666`

## 二、 核心页面结构与交互法则

### 📱 1. 首页：记录 (Home)
**设计核心：** 单手盲操友好，大面积留白，情绪抚慰。

```text
+---------------------------------------------------+
|  [顶部状态栏]                                     |
|                                                   |
|  [随机温情语] 又是用心照顾自己的一天呢～          |
|  [日期] 2026年4月12日                             |
|                                                   |
|  +---------------------------------------------+  |
|  |           [ 当前时段标签组件 ]              |  |
|  |             【 🌞 早餐后 ▼ 】               |  |
|  |   (点击唤起底部 Bottom Sheet 切换其他时段)  |  |
|  +---------------------------------------------+  |
|                                                   |
|  +---------------------------------------------+  |
|  |             [ 核心刻度录入组件 ]            |  |
|  |                                             |  |
|  |       (大点击区，点击唤起数字键盘)          |  |
|  |       [ -0.1 ]     5.5     [ +0.1 ]         |  |
|  |                  mmol/L                     |  |
|  |                                             |  |
|  |       | | | | | | | | | | | | | | |         |  |
|  |         (核心滑动刻度尺，带震动反馈)        |  |
|  +---------------------------------------------+  |
|                                                   |
|  [备注输入框] 吃了什么？或者现在的感受... (选填)    |
|                                                   |
|  [================ 💾 记录血糖 ================]  |
|                                                   |
|  (底部 TabBar: [📍记录]   [📊统计]   [👤我的])    |
+---------------------------------------------------+
```

**关键交互动作：**
* **中心联动：** 中间的刻度尺左右拖动（大范围选值），两侧的 `[-0.1]` / `[+0.1]` 按钮点击（精准步进）。这两者都会实时改变上方的大数字。
* **震动反馈 (Haptic)：** 拖动刻度尺时，每滑动 0.1 个单位，手机必须触发一次极轻微的震动 (`Taro.vibrateShort`)。
* **时间切换 (Bottom Sheet)：** 点击“早餐后 ▼”，屏幕下方平滑弹出一个带遮罩的面板，展示 7 个时段供手动修正。
* **情绪弹窗：** 点击“记录血糖”（附带防连击 Loading），成功后弹出治愈系 Emoji 卡片，停留 2 秒后消失。

### 📱 2. 统计页：数据可视化 (Chart)
**设计核心：** 隐藏干扰线，突出安全区间带，一眼看清趋势。

```text
+---------------------------------------------------+
|  [顶部状态栏]                                     |
|                                                   |
|  [ SegmentedControl 切换: [最近7天] | [最近30天] ] |
|                                                   |
|  +---------------------------------------------+  |
|  |             [ ECharts 图表组件 ]            |  |
|  |  [图例: [空腹] [早餐后] [午餐后] (支持点击隐藏)]|  |
|  |                                             |  |
|  | 10.0 |                                      |  |
|  |      |   /--\           (警戒高位红线)      |  |
|  |  6.7 |---o---o----------o-------------------|  |
|  |      |  /     \        /      [安全绿色背景]|  |
|  |      | /       \      /                     |  |
|  |  4.4 |/---------o----/----------------------|  |
|  |      |                 (警戒低位红线)       |  |
|  |      +--------------------------------------|  |
|  |        4/10    4/11   4/12                  |  |
|  +---------------------------------------------+  |
|                                                   |
|  +---------------------------------------------+  |
|  |               [ 数据简报组件 ]              |  |
|  |  近7天记录: 21 次                           |  |
|  |  完美达标: 18 次 | 偏高: 3 次 | 偏低: 0 次  |  |
|  |  [💡 小贴士: 早餐后波动偏大，建议多散步哦]  |  |
|  +---------------------------------------------+  |
|                                                   |
|  (底部 TabBar: [记录]   [📍📊统计]   [👤我的])    |
+---------------------------------------------------+
```

**关键交互动作：**
* **动态阈值带：** ECharts 背景中的“安全绿色背景”根据用户当前的糖尿病类型（GDM 或 常规）自动调整 Y 轴的高度范围。
* **空状态处理 (Empty State)：** 若无数据，展示一张温暖的插图配文字“图表正在酝酿中... 快去首页记录第一笔数据吧！”。

### 📱 3. 我的：个性化与设置 (Profile)
**设计核心：** 层级极简，操作明确。

```text
+---------------------------------------------------+
|  [顶部背景区 (温馨色调的轻渐变)]                  |
|                                                   |
|    (圆角头像)  微信昵称                           |
|    [ 🤰 孕期控糖模式 (点击切换) ]                 |
|                                                   |
|  +---------------------------------------------+  |
|  |                 [ 功能列表 ]                |  |
|  |                                             |  |
|  |  📊 导出本月数据报表 (Excel)            >   |  |
|  |---------------------------------------------|  |
|  |  ⚙️ 高级设置: 自定义警戒线阈值          >   |  |
|  |---------------------------------------------|  |
|  |  💡 控糖科普与参考标准                  >   |  |
|  |---------------------------------------------|  |
|  |  💬 意见反馈与支持                      >   |  |
|  +---------------------------------------------+  |
|                                                   |
|                                                   |
|  (底部 TabBar: [记录]   [统计]   [📍👤我的])      |
+---------------------------------------------------+
```

**关键交互动作：**
* **自定义阈值页：** 点击进入后，是一个分为 7 个时段的极简表单，允许用户覆盖系统的最高/最低预警线，保证医嘱的绝对优先权。
